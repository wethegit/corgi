# Component styles

All corgi components are styled using [CSS Modules](https://github.com/css-modules/css-modules), which provide safe, component-scoped class names. Below is an example of how CSS modules can be used inside React components.

:::info
If you prefer not to use CSS Modules, you're free to use the CSS workflow of your choice. As mentioned at the start of this section: if it works in Next.js 12, it will work in corgi.
:::

```scss title="/src/components/button/button.module.scss"
.button {
  --icon-size: 1em;

  align-items: center;
  display: inline-flex;
  font-size: 1.2rem;
  padding: 0.5em 3em;
}

.text {
  margin-inline-end: 1em;
}

.icon {
  height: var(--icon-size);
  width: var(--icon-size);
}
```

```jsx title="/src/components/button/button.js"
import * as styles from "./button.module.scss";
import Icon from "../../icon";

const Button = ({ children }) => {
  return (
    <button className={styles.button}>
      <span className={styles.text}>{children}</span>
      <Icon id="arrow-right" className={styles.icon} />
    </button>
  );
};
```
